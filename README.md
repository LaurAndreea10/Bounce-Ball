# Bounce Ball

A browser-based arcade game built as part of a personal front-end portfolio, focused on motion, bounce mechanics, and interactive gameplay.

## Live Demo

https://laurandreea10.github.io/Bounce-Ball/

## Features

* Bounce-based gameplay
* Interactive browser experience
* Dynamic movement mechanics
* Responsive front-end layout
* Simple arcade-style design
* GitHub Pages deployment

## Technologies

* HTML
* CSS
* JavaScript
* GitHub Pages

## Purpose

This project was created as a personal front-end experiment exploring movement, reaction, and arcade-style interaction in a simple web-based game format.
